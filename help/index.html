rplos 0.0-7

PLoS search and ALM from R

Help topics

  • addmissing: Adds elements in a list that are missing because they were not returned
  • alm: Retrieve PLoS article-level metrics (ALM).
  • almdatepub: Get the date when the article was published.
  • almdateupdated: Get the date when article was last updated.
  • almevents: Retrieve PLoS article-level metrics (ALM) events.
  • almplot: Plot results of a call to the alm function.
  • almpubmedcentid: Get PubMed Central article ID by inputting the doi for the article.
  • almpubmedid: Get PubMed article ID by inputting the doi for the article.
  • almtitle: Get title of article by inputting the doi for the article.
  • almtotcites: Alt-metrics total citations from all sources.
  • almupdated: Date when alt-metrics for the article (by DOI) data was last updated.
  • articlelength: Get article length of single paper by DOI, or of many papers as histogram.
  • concat_todf: Concatenate author names, if present, used in other functions.
  • crossref: Lookup article info via CrossRef with DOI.
  • crossref_r: Get random DOI's through CrossRef.
  • formatarticleurl: Format a URL for a specific article in a specific PLoS journal.
  • formatcrossref: Convert crossref XML into a bibentry object
  • getkey: Function to get an API key.
  • journalnamekey: Get short keys for journals to use in searching specific journals.
  • plosabstract: Search PLoS Journals abstracts.
  • plosauthor: Search PLoS Journals authors.
  • plosfields: PLoS API fields to use for searching/retreiving data.
  • plosfigtabcaps: Search PLoS Journals figure and table captions.
  • plossubject: Search PLoS Journals subjects.
  • plostitle: Search PLoS Journals titles.
  • plosviews: Search PLoS Journals by article views.
  • plosword: Search results on a keyword over all fields in PLoS Journals.
  • plot_throughtime: Plot results through time for serach results from PLoS Journals.
  • rplos(rplos-package): Connect with PLoS API data
  • searchplos: Base function to search PLoS Journals

Dependencies

  • Depends: ggplot2
  • Imports: ggplot2, RCurl, RJSONIO, XML, plyr, stringr, reshape, googleVis, httr, grid, gridExtra
  • Suggests: testthat
